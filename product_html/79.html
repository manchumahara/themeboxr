What Airdroop?

Airdroop - is an airdrop listing html template cooked using latest technology like bootstrap 4, less, gulp.

The theme is responsive and displays properly in almost any device.

Airdroop theme is designed thinking airdrop offer, cryptocurrency listing, free coin offer etc in mind and good fit for these purposes.
